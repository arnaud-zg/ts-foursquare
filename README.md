## Getting Started

These instructions will let you know how to use the library.

### Prerequisites

Things you need to install before using this library.

```shell
node: 10.15.0
nvm: 0.33.6
```

### Installing

Using npm:

```shell
npm i --save ts-foursquare
```

Using yarn:

```shell
yarn add --dev ts-foursquare
```

Tests are written with jest
